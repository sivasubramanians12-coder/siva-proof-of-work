No code was pushed to GitHub today. It was a Sunday — a lighter day by design.

What did happen: a new `citation-fixer` skill was added to the BarleyBot agent system. The skill audits and fixes citation and source formatting across the shared Markdown brain — the growing corpus of research, concepts, and reports that feed into the AI system. It runs without inventing sources or deleting unsourced claims, flagging gaps for review instead. A weekly cron was also set up (Sundays, 09:30 IST) to run this automatically.

This is infrastructure work, not feature work — the kind that keeps the knowledge base trustworthy over time. As the brain grows, citation quality becomes a real problem: stale links, missing sources, inconsistent formatting. Fixing it manually doesn't scale. The skill handles it systematically.
