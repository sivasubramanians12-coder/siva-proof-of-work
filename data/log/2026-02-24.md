# Personal AI OS — 3-layer memory working

Cracked the memory architecture today. Three layers:

1. **Episodic** — what happened in this session, stored as timestamped entries
2. **Semantic** — distilled knowledge and patterns, updated after each session
3. **Procedural** — how-to instructions and workflows the agent can call

The problem I kept hitting: agent would lose context mid-session or repeat work it had already done. The fix was separating episodic from semantic memory and running a consolidation step at session end.

Daily digest now pulling from 80+ sources reliably. Voice transcription via Groq Whisper is clean — latency under 2 seconds for typical voice notes.
