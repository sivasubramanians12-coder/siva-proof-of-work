# Launched Proof-of-Work Ledger

Shipped v1 of this site today. The idea: a public, auto-updating record of every business problem I've solved with code.

Five cases live at launch — Geneva, HD Hire, SpiceScience, CoachOS, and Personal AI OS. Each one a real problem from running Hunter Douglas India or from the coaching/knowledge work I do on the side.

The build took one focused session. Stack is Next.js 14 App Router, Tailwind, static generation. No database — data lives in TypeScript files for now, will pull from GitHub API as the ledger grows.

**What's next:** Wire up the GitHub API to pull cases and log entries directly from `sivasubramanians12-coder/siva-builds`. Every commit to that repo should trigger a Vercel rebuild.
